# WTF — Where Talent Forms : SEO Conversion CTA Blocks Report

This report outlines the strategy for integrating high-intent call-to-actions across all informational content.

## 1. 5 Reusable CTA Blocks
We define 5 context-relevant CTA configurations:
1. **CTA Demo**: *"Voir comment WTF peut structurer votre activité"* (pointing to `/demo`).
2. **CTA Apply**: *"Candidater chez WTF"* (pointing to `/apply`).
3. **CTA Atlas**: *"Centraliser votre activité créateur"* (pointing to `/atlas`).
4. **CTA Lex**: *"Préparer vos questions juridiques"* (pointing to `/lex-ai`).
5. **CTA Protection**: *"Protéger votre image et vos accès"* (pointing to `/protection`).

## 2. Page Integration
- Blog articles feature custom `cta` objects rendering an highlighted visual card with these exact headings.
- Landing pages and departments host dedicated custom buttons matching these targets to streamline conversion from cold traffic.
