# Prompts d'Images Hero pour les 20 Articles du Journal WTF

Ce document contient les prompts de génération d'images pour les images hero de chacun des 20 articles du Journal.

## Style Image Global
**Style** : Luxury editorial photography, black couture, ivory, champagne accents, premium creator brand aesthetics. No text, no logos, no watermarks, clean composition, high-end photography lighting, elegant negative space for overlaying titles.
