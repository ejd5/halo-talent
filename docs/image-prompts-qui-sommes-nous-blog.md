## INSTRUCTIONS DE GÉNÉRATION

- **Format :** Utilise --ar 16:9 pour les heroes d'articles, --ar 4:5 pour les slides carrousel, --ar 3:4 pour le portrait histoire, --ar 4:3 pour le bloc couverture
- **Palette :** Toujours mentionner les tons chauds (ambre, or, charbon, ivoire, crème)
- **Style :** "editorial photography", "luxury magazine", "fine art" — éviter le photoréalisme clinique, préférer le film look
- **Post-traitement :** Les images doivent avoir une unité visuelle — tons chauds désaturés, contrastes doux, grain subtil
- **Livraison :** Format WebP ou JPG, 1920px minimum sur le grand côté
