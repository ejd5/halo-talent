# Prompts de Mockups d'Exemples Illustratifs WTF

Ce document liste les prompts de mockups illustratifs anonymisés pour le Journal WTF. Ces captures simulent des interfaces de la suite logicielle WTF, sans utiliser de données réelles ni de logos de plateformes tierces. Chaque mockup comporte la mention discrète "Exemple illustratif".

---

### MOCKUP 01 : Atlas CRM — Segmentation Audience
* **Objectif** : Illustrer la segmentation dynamique des fans.
* **Prompt d'image recommandé** :
  > Premium dark UI dashboard, showing audience segmentation. On the screen, a list of anonymized fan categories (e.g. 'Segment A - High Value', 'Segment B - New Joiners') with clean champagne-gold status dots, simplified graphs showing interaction frequency, a tiny label reading 'Exemple illustratif' in the corner. Luxury minimal UI design, black background, ivory text, 8k.

### MOCKUP 02 : CHATEENG — Brouillon + Validation Humaine
* **Objectif** : Montrer le fonctionnement du copilote de messagerie avec confirmation obligatoire.
* **Prompt d'image recommandé** :
  > Clean chat interface mockup in dark mode. On the left, a simulated incoming fan message. On the right, a highlighted block labeled 'Brouillon généré par l\'IA' containing a draft response, with a prominent golden button labeled 'Valider et envoyer (action humaine requise)'. A small text label saying 'Exemple illustratif' is visible at the bottom. No real platform logos, clean composition, luxury couture interface style.
