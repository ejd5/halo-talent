Halo Companion

Politique de Confidentialité — Extension Chrome

Dernière mise à jour : 10 juin 2026 · Version 1.0.0
✅ Conforme RGPD · Données chiffrées · Consentement explicite

1. Introduction

Halo Companion est une extension Chrome développée par Halo Talent. Cette politique de confidentialité explique quelles données sont collectées, comment elles sont utilisées, et vos droits concernant ces données.

2. Données collectées

2.1 Informations visibles sur les plateformes

L'extension lit uniquement les informations visibles à l'écran sur les plateformes que vous utilisez, avec votre consentement explicite :

  • Profil fan public : nom d'utilisateur, nom d'affichage, badge VIP, montant total dépensé, durée d'abonnement
  • Messages visibles : contenu des messages affichés dans le chat (lecture seule)
  • Statistiques visibles : revenus affichés, nombre de fans, top contributeurs

2.2 Données utilisateur Halo

  • Adresse email (pour l'authentification au service Halo Talent)
  • Préférences de l'extension (thème, langue, plateformes activées, notifications)
  • Notes et tags que vous ajoutez manuellement sur les fans

2.3 Données techniques

  • URL de la page courante (pour détecter la plateforme et la page)
  • Horodatage des actions (pour l'audit log local)
  • Identifiant d'extension Chrome (pour le fonctionnement technique)

2.4 Ce que nous ne collectons JAMAIS

  • Mots de passe ou identifiants de plateforme
  • Médias (photos, vidéos, fichiers) des fans ou du créateur
  • Cookies ou tokens de session des plateformes
  • Données bancaires ou de paiement
  • Messages privés complets — seul un résumé IA (max 200 caractères) peut être généré

3. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Afficher les informations fans dans le side panel de l'extension
  • Générer des suggestions de messages IA pertinentes
  • Fournir des analytics et des insights (LTV, segmentation, taux d'engagement)
  • Synchroniser vos données avec votre compte Halo Talent (si connecté)
  • Journaliser les actions pour votre audit log personnel

4. Stockage des données

  • Stockage local : Toutes les données sont stockées localement dans votre navigateur (chrome.storage.local et IndexedDB)
  • Chiffrement : Les données sensibles sont chiffrées en AES-256-GCM avant stockage local
  • Expiration : Le cache local expire automatiquement après 24 heures (configurable)
  • Backend Halo : Si vous connectez un compte Halo Talent, les données minimales (username, segment, LTV, tags, notes) sont synchronisées avec api.halotalent.com via HTTPS

5. Durée de conservation

  • Cache local : 24 heures par défaut (configurable dans les paramètres)
  • Audit log : 500 derniers événements conservés localement
  • Backend Halo : Aussi longtemps que votre compte Halo Talent est actif

6. Partage des données

Aucune donnée n'est partagée avec des tiers. Les données ne quittent votre navigateur que si vous choisissez de connecter un compte Halo Talent. Même dans ce cas, seules les données minimales nécessaires sont transmises. Les données ne sont jamais vendues, louées, ou utilisées à des fins publicitaires.

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : Consultez toutes vos données dans les paramètres de l'extension
  • Droit de rectification : Modifiez vos tags et notes fan à tout moment
  • Droit à l'effacement : "Effacer le cache local" dans Paramètres → Données supprime tout
  • Droit à la portabilité : "Exporter mes données" dans Paramètres → Données
  • Droit d'opposition : Désactivez l'extension ou des plateformes spécifiques à tout moment
  • Droit de retrait du consentement : Désactivez les plateformes dans Paramètres → Plateformes

8. Sécurité

  • Toutes les données en cache local sont chiffrées avec AES-256-GCM via l'API Web Crypto
  • Les communications avec le backend Halo utilisent HTTPS (TLS 1.3)
  • Le Privacy Guard vérifie chaque donnée avant qu'elle ne quitte le navigateur
  • Le Compliance Guard bloque les actions interdites avant exécution
  • L'extension ne fait PAS de reverse engineering d'API cachée — elle lit uniquement le DOM visible

9. Consentement

L'extension ne fonctionne que sur les plateformes que vous avez explicitement activées dans les paramètres. Aucune lecture de données n'a lieu sans votre consentement préalable. Vous pouvez révoquer ce consentement à tout moment.

10. Contact

Pour toute question concernant cette politique de confidentialité ou vos données :
Email : privacy@halotalent.com
Site web : halotalent.com

Halo Companion

Privacy Policy — Chrome Extension

Last updated: June 10, 2026 · Version 1.0.0
✅ GDPR Compliant · Encrypted Data · Explicit Consent

1. Introduction

Halo Companion is a Chrome extension developed by Halo Talent. This privacy policy explains what data is collected, how it is used, and your rights regarding that data.

2. Data Collected

2.1 Visible Platform Information

The extension reads only on-screen visible information on platforms you use, with your explicit consent:

  • Public fan profile: username, display name, VIP badge, total spent, subscription duration
  • Visible messages: message content displayed in chat (read-only)
  • Visible statistics: displayed revenue, fan count, top spenders

2.2 Halo User Data

  • Email address (for Halo Talent authentication)
  • Extension preferences (theme, language, enabled platforms, notifications)
  • Manually added fan notes and tags

2.3 Technical Data

  • Current page URL (for platform and page detection)
  • Action timestamps (for local audit log)
  • Chrome extension ID (for technical operation)

2.4 What We NEVER Collect

  • Passwords or platform credentials
  • Media (photos, videos, files) from fans or creator
  • Cookies or platform session tokens
  • Payment or banking data
  • Full private messages — only an AI summary (max 200 chars) may be generated

3. Data Usage

Collected data is used exclusively for:

  • Displaying fan information in the extension side panel
  • Generating relevant AI message suggestions
  • Providing analytics and insights (LTV, segmentation, engagement rate)
  • Synchronizing your data with your Halo Talent account (if connected)
  • Logging actions for your personal audit log

4. Data Storage

  • Local storage: All data is stored locally in your browser (chrome.storage.local and IndexedDB)
  • Encryption: Sensitive data is encrypted with AES-256-GCM before local storage
  • Expiration: Local cache automatically expires after 24 hours (configurable)
  • Halo Backend: If you connect a Halo Talent account, minimal data (username, segment, LTV, tags, notes) is synced with api.halotalent.com via HTTPS

5. Data Retention

  • Local cache: 24 hours by default (configurable in settings)
  • Audit log: Last 500 events stored locally
  • Halo Backend: As long as your Halo Talent account is active

6. Data Sharing

No data is shared with third parties. Data only leaves your browser if you choose to connect a Halo Talent account. Even then, only the minimal necessary data is transmitted. Data is never sold, rented, or used for advertising purposes.

7. Your Rights (GDPR)

In accordance with the General Data Protection Regulation (GDPR), you have the following rights:

  • Right of access: View all your data in extension settings
  • Right to rectification: Modify your fan tags and notes at any time
  • Right to erasure: "Clear local cache" in Settings → Data removes everything
  • Right to portability: "Export my data" in Settings → Data
  • Right to object: Disable the extension or specific platforms at any time
  • Right to withdraw consent: Disable platforms in Settings → Platforms

8. Security

  • All locally cached data is encrypted with AES-256-GCM via Web Crypto API
  • Backend communications use HTTPS (TLS 1.3)
  • Privacy Guard validates every piece of data before it leaves the browser
  • Compliance Guard blocks unauthorized actions before execution
  • The extension does NOT reverse-engineer hidden APIs — it only reads visible DOM

9. Consent

The extension only operates on platforms you have explicitly enabled in settings. No data reading occurs without your prior consent. You may revoke this consent at any time.

10. Contact

For any questions about this privacy policy or your data:
Email: privacy@halotalent.com
Website: halotalent.com
