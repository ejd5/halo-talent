## Vos droits fondamentaux

### 1. Vous êtes propriétaire de votre contenu
Dès que vous créez quelque chose (photo, vidéo, texte), vous en êtes automatiquement le propriétaire. Aucun dépôt requis. Votre agence ne peut pas utiliser votre contenu sans votre autorisation.

### 2. Votre compte vous appartient
Les CGU de toutes les plateformes exigent que le compte soit celui d'une personne réelle. Votre agence ne peut PAS être propriétaire de votre compte.

### 3. Vous pouvez vérifier vos revenus
Vous avez le droit de savoir exactement combien la plateforme vous verse. Votre agence doit vous montrer les relevés.

### 4. Vous pouvez partir
Sauf clause de préavis raisonnable (30-60 jours), vous pouvez résilier votre contrat d'agence. Les pénalités excessives peuvent être contestées devant le juge.

### 5. Vos données sont protégées (RGPD)
Si vous êtes en Europe, le RGPD vous donne le contrôle de vos données personnelles et de celles de vos fans.

## Signaux d'alerte

- L'agence insiste pour créer votre email
- L'agence refuse de vous donner accès à vos relevés de revenus
- Le contrat contient des mots comme "à vie", "irrévocable", "exclusivité totale"
- L'agence menace de poursuites si vous voulez partir
- Vous ne pouvez pas contacter directement le support de la plateforme

## Que faire en cas de problème

1. Conservez TOUS les échanges écrits
2. Contactez le support de la plateforme avec votre pièce d'identité
3. Consultez un avocat spécialisé
4. Signalez à la DGCCRF (France) ou équivalent
